# ☁️ Cloud Cost Prediction & Anomaly Detection

## 🚀 Overview
This project predicts cloud infrastructure cost and detects anomalies using machine learning.

It helps identify unusual usage patterns and prevents unexpected cloud billing spikes.

---

## 🧠 Features
- 💰 Cost prediction using Random Forest
- ⚠️ Anomaly detection using Isolation Forest
- 🌐 FastAPI for real-time predictions
- 🎨 Streamlit dashboard for user interaction

---

## 🛠 Tech Stack
- Python
- Scikit-learn
- FastAPI
- Streamlit
- NumPy, Pandas
